TaskBoard is a frontend-only To-Do List web application inspired by Todoist and Apple Reminders. It is built using only HTML, CSS, and Vanilla JavaScript, without any backend or database. The application is fully responsive and works seamlessly on both mobile and desktop devices.

This project was created to practice frontend development fundamentals, responsive UI design, and client-side state management without relying on a backend.

Users can add, edit, delete, and complete tasks. Tasks can be organized into projects, assigned to sections, and optionally given due dates. The app includes search and filtering features, allowing tasks to be viewed as All, Active, or Completed. All data is stored locally in the browser using localStorage, meaning each user‚Äôs data is private to their own device.

Features:
- Responsive design (mobile & desktop)
- Mobile sidebar with hamburger menu
- Project-based task organization
- Add, edit, delete, and complete tasks
- Section and due date support
- Search tasks by title
- Filters: All / Active / Completed
- Data persistence using localStorage (no backend)

Technologies Used:
- HTML5
- CSS3 (Flexbox, Grid, Responsive Design)
- Vanilla JavaScript
- GitHub Pages (deployment)

Project Structure:
The entire application is contained in a single `index.html` file, which includes the HTML structure, CSS styles, and JavaScript logic. No external libraries or frameworks are used, and no installation is required.

How to Run:
- Open the `index.html` file directly in any modern web browser
- Or access the application via the GitHub Pages live demo link

Notes:
- All data is stored in the browser‚Äôs localStorage
- Tasks are not shared across devices or users
- No backend or server-side storage is used by design

This project was developed for educational and portfolio purposes, focusing on clean UI, responsive layout, and core frontend logic.
